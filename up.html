Up

Role / Services

  • Mobile App & Web Development

    • Credits

  • David Nwachukwu

    • Location & year

  • The United Kingdom ©

  • 2020



    • In this solo project, I created a mobile hyper casual game called Up that showcases a range of technical skills and knowledge. I used Unity game engine to develop the game and implemented various programming concepts to create an engaging and challenging gaming experience.


    The concept of the game is simple, yet challenging. The player has to tap the platform carefully to raise the cube up while following the color pattern of the cube and the platform. The cube changes color when on the next platform so that it can go higher, and the player must match the cube's color to the current platform and click on the platform to make the cube rise.


    One of the most complex programming techniques that I implemented in this game is the color pattern matching system. When the player clicks on the platform, the game checks whether the platform's color matches the cube's color. If it does not match, the game ends, and the player has to start over. This system required careful implementation to ensure that the game runs smoothly and accurately tracks the player's progress.


    Another complex programming technique that I implemented is the smooth platform generation system. The game generates new platforms as the player progresses, and these platforms are generated at random positions and colors.


    Furthermore, to make the game more engaging, I implemented a score system that increases with each successful platform click. I also added sound effects and background music to enhance the player's gaming experience.


    Developing Up gave me valuable experiences and knowledge about game development. I learned how to implement complex programming techniques, such as color pattern matching and platform generation, to create a challenging and engaging game. Additionally, I gained experience in game design, balancing, and optimization.


    Overall, Up is an excellent example of my technical skills and creativity as a game developer. The game is optimized for mobile devices and offers a clean and minimalistic design that is easy to navigate. Whether you are a casual gamer or a game enthusiast, I am confident that Up will provide you with an enjoyable and challenging gaming experience.
